### Step 2: Fill Location with Voice

1. **Click "Voice Fill" button** in the Location Information section
2. **Speak your address with structure:**
   - Address/House number
   - Landmark
   - District
   - City
   - Pincode

## üí° Pro Tips

### For Better Title Detection
- Start with the main problem
- Keep it short and clear
- Use "problem is" or "issue is" phrases

### For Better Category Detection
Use these keywords:

| **Want** | **Say** |
|----------|---------|
| Sanitization | "garbage", "waste", "trash", "dustbin" |
| Cleanliness | "dirty", "cleaning needed", "mess" |
| Electricity | "power", "light", "electric", "current" |
| Road | "pothole", "road damage", "street broken" |
| Water | "water leak", "pipe broken", "drainage" |
| Public Safety | "unsafe", "danger", "security issue" |

### For Better Location Detection
- Say "address is" before address
- Say "landmark is" or "near" before landmark
- Speak district name clearly
- Pronounce pincode digit by digit if needed

## üîÑ How to Restart Voice Input

If you made a mistake:

1. Click **"Stop"** button
2. Clear the auto-filled fields manually
3. Click **"Voice Fill"** again
4. Speak again with correct information
